# Проект: Место

### Описание
Проект "Mesto.Russia" - учебный проект от ЯндексПрактикум. Представляет собой сайт c возможностью добавлять фоторграфии, ставить лайки, изменять информацию в профиле пользователя.

## Использованные технологии:
* Методология БЭМ
* Nested
* Флексбокс
* Позиционирование
* JavaScript
